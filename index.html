網址參數說明

你可以在網址後面加上三個參數，開啟網頁時自動帶入 API 提供者、API Key 和模型。

provider
可填 cgu、openai、google、claude
apiKey
你的 API Key
model
模型名稱，例如 gpt-5.4-mini

注意：API Key 放在網址中可能會留在瀏覽器紀錄或分享連結裡，建議只在自己可信任的裝置上使用。
