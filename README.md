# Simple Knowledge Chatbot

Pure frontend knowledge-base chatbot for GitHub Pages.

## Features

- Upload `.txt` and `.md` files in the browser
- Split and retrieve relevant knowledge chunks locally
- Choose CGU, OpenAI, Google AI Studio, or Claude API providers
- Use preset models or type a custom model name
- Collapse the settings panel for a wider chat layout

API keys and uploaded knowledge files are not stored in `localStorage`.
